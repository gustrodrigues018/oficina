Atividade Prática

Exercício 1: Organizando sua Pasta de Estudos

Vamos colocar em prática o que aprendemos sobre gerenciamento de arquivos!

  • Crie uma pasta na área de trabalho: "Meus Estudos TI"
  • Dentro dela, crie 3 subpastas: "Aula 1 - SO", "Aula 2 - Internet", "Projetos"
  • Crie um arquivo de texto simples (Bloco de Notas) e salve na pasta "Aula 1 - SO"
  • Pratique copiando e movendo o arquivo entre as pastas
  • Renomeie o arquivo para "Minha Primeira Atividade"

Dica Importante: Sempre mantenha seus arquivos organizados. Uma boa organização economiza muito tempo no futuro!

Exercício 2: Explorando Atalhos

Teste os atalhos de teclado que você aprendeu e sinta a diferença na agilidade!

  • Abra um documento de texto ou o Bloco de Notas.
  • Digite algumas frases.
  • Use Ctrl + C e Ctrl + V para copiar e colar texto.
  • Use Ctrl + X e Ctrl + V para recortar e colar texto.
  • Use Ctrl + Z para desfazer uma ação.
  • Use Ctrl + S para salvar seu documento.
  • Experimente Alt + Tab para alternar entre programas.

Desafio: Tente fazer todas as operações sem usar o mouse!
